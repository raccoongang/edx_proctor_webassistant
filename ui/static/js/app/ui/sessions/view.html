В поле Центр тестирования ведите условное обозначение для вашего центра и сессии прокторинга.
Старайтесь соблюдать единый формат, это поможет вам быстро найти нужную сессию в списке.
Рекомендуемый формат:
ВашЦентрТестирования, Аудитория, ФамилияПроктора, Дата(ГГ-ММ-ДД), Время(ЧЧ:ММ)
Пример:
МФТИ-КПМ, АУД.12, Иванов, 2015-12-20, 09:00
